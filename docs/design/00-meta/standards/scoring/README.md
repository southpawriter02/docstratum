## Overview

The quality scoring framework defines how the 30 validation criteria contribute to a composite 0–100 quality score. It specifies three dimension weights (Structural 30%, Content 50%, Anti-Pattern 20%), five grade thresholds, and one gating rule (critical anti-patterns cap the total at 29).
